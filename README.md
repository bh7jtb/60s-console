# 60s-console
# 60s API 信息聚合控制台

Dashboard 风格的信息聚合页面，参考官方设计稿实现。

## 功能特点

- **顶部导航**：首页 / 热榜 / 新闻 / 天气 / 工具 五大板块切换
- **搜索栏**：支持按关键词快速查找接口
- **卡片式布局**：响应式网格，自适应桌面和移动端
- **深色/浅色主题**：一键切换明暗模式
- **5种主题色**：松绿、仓蓝、珊瑚、藤紫、金色
- **API 地址配置**：可在设置面板中修改并持久化保存
- **自动加载**：首页默认加载今日60秒、城市天气、必应壁纸、抖音热搜

## 覆盖的 API 接口

### 🕘 周期资讯（首页）
- 每天60秒读懂世界、AI资讯快报、必应每日壁纸、货币汇率、历史上的今天、Epic免费游戏、实时IT资讯

### 🔥 热门榜单（热榜页）
- 抖音热搜、微博热搜、B站热搜、知乎话题、头条热搜、小红书热点、懂车帝、百度热搜、夸克热点、Hacker News

### 🌤️ 天气（天气页）
- 实时天气、天气预报（支持城市切换）

### 🛠️ 实用工具（工具页）
- 在线翻译(109种语言)、歌词搜索、二维码生成、百度百科、公网IP、Whois查询、黄金价格、汽油价格、摸鱼日报、身体分析、密码生成器、随机颜色等
- 娱乐消遣：一言、搞笑段子、随机运势、答案之书、KFC文案、冷笑话

## 部署方式

```bash
# 1. 安装 wrangler
npm install -g wrangler

# 2. 登录 Cloudflare
wrangler login

# 3. 进入项目目录并部署
cd projects/60s-console
wrangler deploy
